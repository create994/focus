MAX Productivity Bot

This chatbot mini-application connects universities and organizations with their audiences through the MAX messenger.

Capabilities

  • Interactive chat commands for schedules and reminders
  • Automatic event reminders using MAX platform APIs
  • Support for Russian and English language interfaces

API Endpoints

  • GET /api/events — list upcoming events
  • POST /api/subscribe — subscribe user to events or categories
  • POST /api/chat — send a bot command manually
  • POST /api/webhook — entry point for MAX messenger webhook
